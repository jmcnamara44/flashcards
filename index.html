JavaScript (JS): is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm. Alongside HTML and CSS, JavaScript is one of the three core technologies of World Wide Web content engineering

Operators: JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Variables: JavaScript variables are containers for storing data values.

Variable naming conventions: Coding conventions are style guidelines for programming

Functions: JavaScript function is a block of code designed to perform a particular task

Methods: JavaScript methods are the actions that can be performed on objects

Arguments: Function arguments are the real values received by the function when it is invoked. Inside the function, the arguments are used as local variables

Parameters: Parameters are variables listed as a part of the function definition

Return: he return statement stops the execution of a function and returns a value from that function.

Chaining Methods: Chaining allows us to run multiple jQuery methods (on the same element) within a single statement
